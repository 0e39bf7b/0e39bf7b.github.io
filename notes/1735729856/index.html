The Linux kernel has traditionally been compiled using GCC and binutils. However, it is now possible to build the Linux kernel with Clang. Distributions such as Android, ChromeOS, OpenMandriva, and Chimera Linux already use Clang-built kernels. ClangBuildLinux is an effort to get the Linux kernel to compile with Clang.
